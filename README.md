## Project Description

This project is a Node.js API server that calculates health insurance risk based on user input.
The API processes data such as age, BMI, blood pressure, and family history to determine a
risk score and category.

The server is designed as part of a client-server architecture and is deployed using
Microsoft Azure App Service.

## How to Run the Application

1. Clone the repository  
2. Navigate to the project folder  
3. Install dependencies:  
   npm install  
4. Start the server:  
   node server.js  
5. Open browser or Postman and go to:  
   http://localhost:3000  

---

## API Endpoints

1. Ping (Wake-Up)  
   GET /api/ping  

2. BMI  
   POST /api/bmi  

3. Blood Pressure Category  
   POST /api/bp-category  

4. Risk Category  
   POST /api/risk-category
